WhatsApp Media Player

Installation

On the Chrome web store or

  • clone this repo git checkout git@github.com:select/whatsapp-web-media-player.git
  • Install dependencies with npm install
  • Build with npm run sync and npm start
  • chrome://extensions/ click on "Load unpacked extension"
  • reload on extension page with ctrlr
